This CD-ROM accompanies N. S. Nise, Control Systems 
Engineering, Third Edition, Wiley, New York, NY, 2000. 
It contains ancillary matter to enrich the readers of 
Control Systems Engineering. Specifically it contains 
additional appendixes, computer files, graphics files, 
and solutions to Skill-Assessment Exercises. Appendixes 
are to be considered an extension of the appendixes in 
the book. Hence, some appendixes on this disk are 
lettered sequentially following the last appendix in 
the book.

In order to use this disk, you must have a PC with 
Windows 95 or later installed. In addition, other 
software described below for individual appendixes or 
files must be installed. See installation instructions 
within the README files in the individual folders.

for their instructional value. They have been tested

C. DISK ORGANIZATION

The following is the contents and folder organization 
of the disk:

1. Appendixes Folder: Contains additional appendixes

Appendix G   Matrices, Determinants, and Systems of Equations
Appendix H   Control System Computational Aids
Appendix I   Solutions of State Equations for t0 not = 0
Appendix J   Root Locus Rules: Derivations

2. Control Systems Engineering Toolbox folder: Contains 
executable MATLAB based programs referenced in the text.

Chapter  4 Supplement  Simulink Files for Selected Examples
Appendix B Supplement  MATLAB Files for Appendix B
Appendix C Supplement  Simulink Files for Appendix C
Appendix D Supplement  MATLAB Files for Appendix D
Appendix E Supplement  MATLAB Files for Appendix E

3. Lecture Graphics: Contains selected figures from Control 
Systems Engineering, 3rd ed. in both pdf and PowerPoint 
formats suitable for creating projected images for use 
during class.

4. Solutions to Skill-Assessment Exercises: Contains
solutions to Skill-Assessment Exercises in the text. 
 
5. Adobe Acrobat Reader: Contains the Adobe Acrobat Reader
software, which must be installed in order to read the pdf
files contained on this CD-ROM.


D. REQUIRED SOFTWARE 

The following software is required in order to use files 
within the specified folder:

1. Chapter 4 Supplement: MATLAB Version 5 and MATLAB's 
Simulink Version 3.

2. Appendix B Supplement: MATLAB Version 5 and MATLAB's 
Control System Toolbox Version 4. 

3. Appendix C Supplement: MATLAB Version 5 and MATLAB's 
Simulink Version 3.

4. Appendix D Supplement: MATLAB Version 5 and MATLAB's 
Control System Toolbox Version 4.2. 

5. Appendix E Supplement: MATLAB Version 5, MATLAB's 
Control System Toolbox Version 4, and MATLAB's Symbolic 
Math Toolbox Version 2. 

6. Appendix G: Adobe Acrobat Reader

7. Appendix H: Adobe Acrobat Reader. Computer programs 
contained within this folder require Windows 95.

8. Appendix I: Adobe Acrobat Reader

9. Appendix J: Adobe Acrobat Reader

10. Lecture Graphics: Adobe Acrobat Reader or 
Microsoft PowerPoint

11. Solutions to Skill-Assessment Exercises: Adobe 
Acrobat Reader
